---
description: How to export the manuscript to a single file
---

1. Identify all drafted chapters and front-matter/back-matter files in the correct order.
2. Concatenate the contents of these files into a single markdown file.
3. Save the resulting file to the `exports/` directory with a timestamped or versioned filename.
4. If specifically requested, convert the markdown to other formats (PDF, EPUB) using external tools if available.
